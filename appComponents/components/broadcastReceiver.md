# Broadcast Receiver

Позволяет реагировать на состояния вне приложения 

К примеру на отсутствие сети или низкого заряда батареи 

```kotlin
class BatteryReceiver : BroadcastReceiver() {
  override fun onReceive(p0: Context?, p1: Intent?) {

  }
}
```
